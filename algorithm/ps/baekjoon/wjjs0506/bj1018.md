# 체스판 다시 칠하기

## 문제정보

- 사이트 : boj(백준 알고리즘 사이트)
- 문제 번호 : 1018
- 문제 분류 : 브루트포스
- 난이도 : silver 3

## 문제 풀이

### 생각한 문제 조건

> **💬 생각 노트**  
> N~M 범위 내의 맨 왼쪽 좌표의 시작 지점이 B,W 2케이스의 변화하는 count의 최소 값을 탐색한다.

### 내가 짠 코드

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class BJ1018 {

    static String[][] chessB = {
            {"B","W","B","W","B","W","B","W"},
            {"W","B","W","B","W","B","W","B"},
            {"B","W","B","W","B","W","B","W"},
            {"W","B","W","B","W","B","W","B"},
            {"B","W","B","W","B","W","B","W"},
            {"W","B","W","B","W","B","W","B"},
            {"B","W","B","W","B","W","B","W"},
            {"W","B","W","B","W","B","W","B"}
    };

    static String[][] chessW = {
            {"W","B","W","B","W","B","W","B"},
            {"B","W","B","W","B","W","B","W"},
            {"W","B","W","B","W","B","W","B"},
            {"B","W","B","W","B","W","B","W"},
            {"W","B","W","B","W","B","W","B"},
            {"B","W","B","W","B","W","B","W"},
            {"W","B","W","B","W","B","W","B"},
            {"B","W","B","W","B","W","B","W"}
    };

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String[] input = br.readLine().split(" ");
        int N = Integer.parseInt(input[0]);
        int M = Integer.parseInt(input[1]);

        String[][] map =  new String[N][M];

        // map 형식으로 세팅
        for (int i = 0; i < N; i++) {
            input = br.readLine().split("");
            for (int j = 0; j < M; j++) {
                map[i][j] = input[j];
            }
        }

        int x=0, y=0;
        int minCount=Integer.MAX_VALUE;

        while(x + 8 <= N && y + 8 <= M) {
            // 첫 시작이 B일 경우, 변경 횟수 체크
            int tempB = checkMinB(map,x,y,minCount);
            if(tempB<minCount) {
                minCount=tempB;
            }
            // 첫 시작이 W일 경우, 변경 횟수 체크
            int tempW = checkMinW(map,x,y,minCount);
            if(tempW<minCount) {
                minCount=tempW;
            }
            y++;
            // 증가한 x가 N이 넘어가면 y값 증가 후 x 0으로 초기화
            if(y + 8 > M) {
                x++;
                y=0;
            }
        }

        System.out.println(minCount);
    }

    public static int checkMinB(String[][] map, int x, int y, int count){
        int tempC=0;
        for (int i = x; i < x+8; i++) {
            for (int j = y; j < y+8; j++) {
                if(!map[i][j].equals(chessB[i-x][j-y])) {
                    tempC++;
                    // 현재 최소 값 넘어갈 경우 중지
                    if(tempC>count) {
                        return Integer.MAX_VALUE;
                    }
                }
            }
        }
        return tempC;
    }

    public static int checkMinW(String[][] map, int x, int y, int count){
        int tempC=0;
        for (int i = x; i < x+8; i++) {
            for (int j = y; j < y+8; j++) {
                if(!map[i][j].equals(chessW[i-x][j-y])) {
                    tempC++;
                    // 현재 최소 값 넘어갈 경우 중지
                    if(tempC>count) {
                        return Integer.MAX_VALUE;
                    }
                }
            }
        }
        return tempC;
    }
}
```

## 의문점 및 개선 한 코드

내가 구현한 코드는 흑인 경우의 체스판과 백인 경우의 체스판을 선언해서 진행하였지만, 2차원 배열의 각 좌표(i,j)의 합이 짝수인지 홀수인지로 해당 조건을 간소화 할 수 있다.

아래와 같은 방법으로 구현 시 반복적인 함수 내용을 간소화 할 수 있다.

```java
int countW = 0; // 시작 값이 W일때 카운트
int countB = 0; // 시작 값이 B일때 카운트

// 탐색(1번)
for (int i = x; i < x + 8; i++) {
    for (int j = y; j < y + 8; j++) {
        if ((i + j) % 2 == 0) {
            if (graph[i][j] != 'W') countW++;
            if (graph[i][j] != 'B') countB++;
        }
        if ((i + j) % 2 != 0) {
            if (graph[i][j] != 'B') countB++;
            if (graph[i][j] != 'W') countW++;
        }
    }
}
```
