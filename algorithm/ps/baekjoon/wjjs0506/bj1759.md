# 제목

## 문제정보

- 사이트 : boj(백준 알고리즘 사이트)
- 문제 번호 : 1759
- 문제 분류 : 백트래킹
- 난이도 : gold 5

## 문제 풀이

### 생각한 문제 조건

1. 자료형 제한 조건
   - 범위가 3 <= L <= C <= 15 이므로 재귀의 깊이가 문제되진 않는다.
2. 문제의 다른 조건
   > **💬 생각 노트**  
   > 전형적인 백트래킹 문제로 선택했을 경우와 선택 안했을 경우로 나눠 재귀를 태웠다.  
   > 암호의 길이만큼 선택했을 때, 자음과 모음의 갯수를 체크하여 출력 여부를 결정하여도 되지만, 선택했을 때 자음 모음 여부를 판단하여 각 갯수를 매개변수로 줘 진행하였다.  
   > 재귀의 활용에 익숙해지기위해 선택한 방법이지만, 해당 방법은 매 재귀마다 검사하므로 전자의 방법이 더 효율적이다.

### 내가 짠 코드

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Arrays;

public class BJ1759 {
    static int L = 0;
    static String[] vowels = {"a", "e", "i", "o", "u"};

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String[] input = br.readLine().split(" ");
        StringBuilder sb = new StringBuilder();
        L = Integer.parseInt(input[0]);

        input = br.readLine().split(" ");

        // 정렬된 문자열 암호
        Arrays.sort(input);

        int start = 0;
        solve(input, start, 0, 0, sb);
    }

    public static void solve(String[] word, int index, int vowelC, int consonantC, StringBuilder result) {
        // 기저 조건 : 선택한 것이 암호 길이보다 크거나 모든 원소를 확인했을 때
        if (result.length() == L || index == word.length) {
            // 모음 1개 자음 2개를 만족 하는 경우
            if(result.length() == L && vowelC >= 1 && consonantC >= 2) {
                System.out.println(result);
            }
            return;
        }

        // 선택 한 경우
        result.append(word[index]);
        if(Arrays.asList(vowels).contains(word[index])) {   // 모음인 경우
            solve(word, index + 1, vowelC+1, consonantC, result);
        } else {    // 자음인 경우
            solve(word, index + 1, vowelC, consonantC+1, result);
        }

        // 선택하지 않은 경우
        result.deleteCharAt(result.length()-1);
        solve(word, index + 1, vowelC, consonantC, result);
    }
}
```
