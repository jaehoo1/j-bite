## 문제정보

- 문제 번호 : 2798
- 문제 분류 : 브루트포스
- 난이도 : bronze 2

## 문제 풀이

### 생각한 문제 조건

1. 자료형 제한 조건  
   -. N이 3 이상 100미만이므로 n^3을 해도 100만 이므로 브루트 포스로 진행해도 괜찮다. (3중 for문)

   > **💬 생각 노트**  
   > 무식하게 전 경우의 수에서 m보다 작으면서 전체 값중 최대값을 찾는다.  
   > 3중 for문을 통해 j,k의 범위를 각각 i+1, j+1부터 시작하여 전체 3항의 합을 구한다.

### 내가 짠 코드

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class BJ2798 {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String[] input = br.readLine().split(" ");
        int n = Integer.parseInt(input[0]);
        int m = Integer.parseInt(input[1]);

        int[] numArr = new int[n+1];
        input = br.readLine().split(" ");
        for (int i = 0; i < n; i++) {
            numArr[i] = Integer.parseInt(input[i]);
        }

        int max = 0;
        int temp = 0;

        for (int i = 0; i < n; i++) {
            for (int j = i+1; j < n; j++) {
                for (int k = j+1; k < n; k++) {
                    temp = numArr[i] + numArr[j] + numArr[k];
                    if(temp <= m && temp > max){
                        max = temp;
                    }
                }
            }
        }
        System.out.println(max);
    }
}
```
